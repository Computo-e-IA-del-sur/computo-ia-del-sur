# Guía de Identidad de Marca: Computo e IA del Sur

## 1. Concepto Creativo: "El Vector del Sur" (The South Vector)

La identidad visual de **Computo e IA del Sur** se basa en la convergencia de tres pilares fundamentales: **Geometría Tecnológica**, **Redes de Inteligencia** y **Dirección Estratégica (Sur)**.

El símbolo seleccionado combina una brújula abstracta con una arquitectura de red neuronal. La forma apunta hacia abajo (Sur) simbolizando el origen, el territorio y una visión firme, mientras que su estructura interna de nodos interconectados representa el procesamiento de datos y la inteligencia artificial.

### Significado del Símbolo
- **Forma Diamante/Brújula:** Representa precisión, valor y dirección. El eje vertical enfatiza la orientación hacia el sur.
- **Nodos Interconectados:** Simbolizan la red neuronal, la conectividad y el aprendizaje profundo.
- **Espacio Negativo:** Sugiere apertura y escalabilidad.

## 2. Paleta de Colores

Se ha seleccionado una paleta moderna, vibrante y puramente tecnológica que se aleja de los tonos genéricos de la industria.

| Color | Hex | Aplicación |
| :--- | :--- | :--- |
| **Cian Tecnológico** | `#06B6D4` | Color principal, representa innovación y claridad. |
| **Violeta IA** | `#8B5CF6` | Color secundario, representa inteligencia y futuro. |
| **Amarillo Energía** | `#FACC15` | Color de acento, representa dinamismo y luz. |
| **Gris Carbón** | `#111827` | Fondos oscuros y texto principal. |
| **Blanco Frío** | `#F8FAFC` | Fondos claros y contraste. |

### Degradado de Marca
**Neural Flow Gradient:** `linear-gradient(135deg, #06B6D4 0%, #8B5CF6 100%)`

## 3. Tipografía

**Principal:** `Geist Sans` (implementada vía Next.js font).
Es una tipografía sans-serif diseñada para interfaces tecnológicas, con un balance perfecto entre legibilidad y modernidad.

## 4. Versiones del Logotipo

- **Principal (Horizontal):** Símbolo + Texto. Para web y documentos oficiales.
- **Símbolo:** Uso en avatares, favicons y elementos donde el espacio es reducido.
- **Vertical:** Para banners o aplicaciones donde el eje vertical es predominante.
- **Monocromático:** Versiones en blanco o negro para aplicaciones limitadas.

## 5. Implementación Técnica

Los activos se encuentran en `/public/brand/` en formato SVG optimizado para garantizar escalabilidad infinita y carga rápida.

- `logo.svg`: Versión completa a color.
- `logo-dark.svg`: Optimizado para fondos oscuros.
- `logo-icon.svg`: Símbolo independiente.
- `favicon.svg`: Icono de pestaña del navegador.

## 6. Usos Correctos e Incorrectos

**Correcto:**
- Mantener el área de seguridad alrededor del logo (mínimo el ancho del símbolo).
- Utilizar la versión de contraste adecuada según el fondo.

**Incorrecto:**
- Estirar o deformar el logo.
- Cambiar los colores de la paleta oficial.
- Utilizar el logo sobre fondos con demasiado ruido visual.
