Política de Privacidad - Reportes Tren Ligero

Última actualización: Julio 2026

En Cómputo EIA del Sur, nos tomamos muy en serio la privacidad de nuestros usuarios. Esta política describe cómo manejamos la información en la aplicación móvil Reportes Tren Ligero.

1. Información que recopilamos

Nuestra aplicación recopila reportes de estado del servicio en tiempo real introducidos voluntariamente por los usuarios (como retrasos, afluencia o fallas). No vendemos ni compartimos datos personales con terceros.

2. Permisos del Dispositivo

La aplicación puede solicitar acceso a la ubicación del dispositivo únicamente con el fin de validar que el reporte del usuario corresponda a una estación cercana del Tren Ligero. Estos datos no se almacenan de forma permanente en nuestros servidores.

3. Seguridad

Implementamos medidas de seguridad técnicas para proteger la información de los reportes enviados a nuestra base de datos (Supabase).

4. Contacto

Si tienes dudas sobre esta política, puedes contactarnos en nuestro sitio web oficial: computoeiadelsur.com

5. Formulario de contacto del sitio web

Además de la aplicación móvil Reportes Tren Ligero, este sitio web (computoeiadelsur.com) incluye un formulario de contacto para solicitudes de cotización. Los datos que envíes ahí (nombre, correo electrónico y el mensaje) se utilizan únicamente para responder tu solicitud y no se venden ni se comparten con terceros, salvo con el proveedor que procesa el envío del formulario (Formspree) únicamente para hacernos llegar tu mensaje.
